Initial public surface: framework-agnostic store + `toast()` facade, `toast.promise` lifecycle, React `<Toaster/>` / `<ToastItem/>` with six positions, stacking and queue, pause-on-hover/blur, swipe-to-dismiss, FLIP stack reflow, a documented `aria-live` accessibility model, and a headless render path alongside the styled default. Dual ESM/CJS, types included, zero runtime dependencies.
