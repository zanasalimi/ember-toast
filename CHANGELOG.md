# Changelog

This file aggregates release notes across the workspace. Per-package changelogs live in `packages/*/CHANGELOG.md` and are generated by [changesets](https://github.com/changesets/changesets) on release. Do not edit released entries by hand — add a changeset instead (`pnpm changeset`).

## Unreleased

### Added
- Framework-agnostic core store: pub/sub, id allocation, queue with max-visible overflow, per-toast timers with pause/resume.
- `toast()` facade: `success`, `error`, `warning`, `info`, `loading`, `custom`, plus `dismiss(id)` / `dismiss()` and update-by-id.
- `toast.promise()` lifecycle that morphs a single toast in place through loading → success → error.
- React binding: `useToasts` (via `useSyncExternalStore`), `<Toaster/>`, `<ToastItem/>`.
- Six positions, configurable stack direction, pause-on-hover and pause-on-blur.
- Swipe-to-dismiss (pointer/touch) with velocity threshold and snap-back.
- FLIP-based stack reflow so removals slide rather than jump.
- Accessibility model: single `aria-live` region, `role` by severity, no focus theft, `prefers-reduced-motion` honored.
- Headless render path plus a styled default themed through CSS custom properties.
- Dual ESM/CJS build via tsup with emitted `.d.ts`; `sideEffects` configured for tree-shaking.
