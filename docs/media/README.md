# Media

`hero.png` — the showcase image used at the top of the root [`README.md`](../../README.md).
It renders the styled `<Toaster/>` default across every variant (success, error,
info with a determinate progress bar, warning, loading) in both the dark and light
themes, so the card design reads at a glance without a screen recording.

For motion — FLIP reflow, promise morph, swipe-to-dismiss — use the live
[playground](https://zanasalimi.github.io/ember-toast/); it dogfoods the real package.
